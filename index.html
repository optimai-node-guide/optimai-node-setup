Troubleshooting

Common Problems & Fixes

Running into an issue? Here are the most common problems and how to solve them quickly.

🔋 Battery draining too fast on mobile?

Go to Battery Settings → App Battery Usage → OptimAI Edge Node → Set to Restricted or Optimized. This limits background activity while keeping the node connected.

📶 How much internet data does OptimAI use?

OptimAI Node uses minimal data — typically a few MB per day. It primarily uses idle processing power, not heavy bandwidth. WiFi is recommended.

🛑 Node keeps stopping on Android?

Go to Settings → Apps → OptimAI → Battery → Allow background activity. Also disable battery optimization for the app.

💻 Can I run multiple devices on one account?

Yes. You can run OptimAI Node on your Mobile App, Telegram Lite Node, and Desktop PC simultaneously. All rewards accumulate together in your dashboard.

🔒 Is OptimAI Network safe to use?

OptimAI uses only your device's idle computing resources. It does not access personal files or sensitive data. Always download from Google Play Store or Apple App Store only.
